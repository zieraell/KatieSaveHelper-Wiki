This is a utility mod for ***ENA: Dream BBQ*** focused on improving the speedrunning experience, both during and outside of runs.

This project was originally created to maintain the mod *"Katie's Weird Practice Save Helper"* by ***@Katelyndev2011***, and has since been expanded into a much larger mod.

See [*Getting Started*](getting-started/index.md) for more info about the mod's features and how to properly install it.
