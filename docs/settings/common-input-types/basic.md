### Boolean

Can be either *true* or *false*. Typically, *true* enables the behaviour described in the setting, while *false* disables it.

### Integer

Can be any *whole number* (ex. 1, 2, 3) between around negative 2 billion to positive 2 billion.

### Float

Can be any *decimal number* (ex. 1, 1.5, 1.05) between around negative 340 undecillion to positive 340 undecillion.

### Hex Code

Can be any *hex code*, in either RGB (ex. #RRGGBB) or RGBA (ex. #RRGGBBAA) format. The *'#'* symbol is not required.
