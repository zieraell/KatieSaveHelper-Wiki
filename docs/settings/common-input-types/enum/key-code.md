---

An enum used by settings that bind the mod's various *hotkey actions* to keyboard or mouse inputs.

<br>

| Category | Valid values |
|-----------|--------------|
| Number keys | *Alpha0* → *Alpha9* |
| Keypad | *Keypad0* → *Keypad9* |
| Function keys | *F1* → *F12* |
| Letters | *A* → *Z* |
| Mouse buttons | *Mouse0* → *Mouse6* |
| Common keys | *Space*, *Escape*, *Tab*, *Backspace*, *LeftShift*, *RightControl*, etc. |
| Symbols | *Exclaim*, *DoubleQuote*, *Hash*, *Dollar*, *Percent*, *Ampersand*, *Quote*, *LeftParen*, *RightParen*, *Asterisk*, *Plus*, *Comma*, *Minus*, *Period*, *Slash*, *Colon*, *Semicolon*, *Less*, *Equals*, *Greater*, *Question*, *At*, *LeftBracket*, *Backslash*, *RightBracket*, *Caret*, *Underscore*, *BackQuote* |
| Special | *None* ***(disables the hotkey)*** |
